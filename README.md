## 📝 To-Do List Application (Python GUI):

A simple and beginner-friendly To-Do List desktop application built using Python and Tkinter.
This app allows users to add, delete, and manage daily tasks with automatic saving — so tasks remain even after closing the program.

---

## ✨ Features:

- Add new tasks
- Delete selected tasks
- Clear all tasks
- Auto-save tasks to file
- Load saved tasks on startup
- Warning popups for empty inputs
- Simple and clean graphical interface

---

## 🖥️ Tech Used:

- Python
- Tkinter (built-in GUI library)
- File Handling (text file storage)

---

## 📂 Project Structure:

- todo-list/
- │── to_do_list.py
- │── tasks.txt   (auto-created)
- │── README.md

---

## 💾 How It Works:

- Tasks entered by user are saved in tasks.txt
- When the app restarts → tasks automatically reload
- No data loss after closing the application
